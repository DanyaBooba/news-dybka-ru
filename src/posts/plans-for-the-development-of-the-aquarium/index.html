План развития Аквариума

В основной статье мы говорили о новом и интересном проекте Аквариум.

Теперь пришла пора рассказать о том, как этот проект будет развиваться. Постараемся максимально ясно рассказать о планах развития проекта, и привести примеры.

Новый движок

Первое обновление, над которым происходит работа уже сейчас — переход на новый движок Laravel.

У меня уже был печальный опыт работы с движком Laravel, поэтому когда я начал создавать проект Аквариум, я не планировал внедрять в него какой-либо движок.

Но, к сожалению, уже сейчас я понимаю, что дальше разрабатывать проект в таком формате у меня не выйдет. На данный момент проект разрабатывается на чистом PHP. Пусть я и применяю различные удобные модули, чтобы разработка проекта шла быстрее, в течением времени проект будет все тяжелее, и его будет все сложнее разрабатывать.

Для того, чтобы добиться успеха, нужно обязательно масштабировать проект. И если мы говорим о разработке на чистом PHP, то масштабирование проекта с каждым разом будет все сложнее и будет занимать все больше времени.

Данное решение нельзя оставлять в таком формате, именно поэтому проект Аквариум начинает переходить на движок Laravel.

Посты, фотографии и социальное взаимодействие

Следующий шаг, который мы совершим — добавление новых возможностей для пользователей. И прежде всего, будут добавлены посты в открытом виде, будет добавлена возможность добавлять фотографии в соцсети.

И также мы продолжим развивать социальное взаимодействие между пользователями. Новый движок Laravel позволит нам решить большое количество проблем, именно поэтому мы без проблем сможем внедрить новые решения для пользователей.

Приложения

Ещё одно важное решение для социальной сети — удобные приложения для разных платформ. Прежде всего, мы разработаем мобильное приложение для операционной системы Андроид.

Приложение для операционной системы Андроид уже разрабатывается на данный момент. После разработки у пользователей будет полный функционал, как и в веб версии приложения, но только локально на устройстве пользователя.

Далее будут разработаны приложения для операционных систем iOS, macOS, Windows и Linux. Мы постараемся обеспечить удобством всех пользователей

OAuth авторизация

Мы добавим возможность интегрировать вход в аккаунт с помощью технологии OAuth для Аквариума. У пользователей появится возможность входить в аккаунт в два клика с помощью технологии OAuth Аквариума.

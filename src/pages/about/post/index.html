Добавить статью

На данной странице расположена инструкция, как правильно оформить статью к публикации на сайте Special. К сожалению, на данный момент на сайте Special нет единой формы публикации статьи, но поверьте, в публикации статьи нет ничего сложного.

Следуйте инструкции на данной странице, чтобы успешно опубликовать статью на сайте Special.

Написание статьи

Для написания статьи вы можете использовать любой удобный редактор текста. Создать изображения для статьи вы можете самостоятельно, либо с помощью нейросети. О правилах оформления, размерах и прочего поговорим позже.

Проверка содержания статьи

Перед отправкой статьи на модерацию, вам следует проверить ее содержание. Убедитесь, что:

  • статья не нарушает законодательство РФ;
  • в статье нет описания опасных действий;
  • цель статьи — информирование, распространение незапрещенной информации, но не пропагандирование или призыв к действию;
  • текст статьи не содержит оскорблений;
  • статья подходит для всех возрастов.

В исключительных случаях модератор, проверяющий вашу статью, может отказать в публикации статьи по иным причинам. Пожалуйста, будьте к этому готовы.

Желательно, перед публикацией статьи, посмотреть, какие статьи уже были опубликованы на сайте Special. Так вы сможете ориентироваться, как вам следует оформить статью.

Файл index.html

Подготовим файл index.html. В данном файле будет содержаться статья. Используйте следующий исходный код для формирования статьи.

Открыть файл

Используйте следующие правила:

  • вместо «<заголовок>» напишите заголовок статьи (до 90-ста символов);
  • вместо «<описание_изображения>» напишите описание изображения (требуется для людей, использующих программы Screen Reader);
  • вместо «<блок_текста>» напишите текст статьи, используйте несколько раз тег <p>, чтобы разделить текст на абзацы.

Структура папки со статьей, изображение статьи

В данном блоке поговорим о том, как должна выглядеть статья перед публикацией. Проверьте, что вы подготовили следующие файлы: index.html (основная статья), изображение (jpg, 1200x600), иные изображения.

Назовем папку, которая будет содержать все эти файлы — проектом. Название проекта должно быть следующего формата: английские буквы, разделитель: «-», только прописные буквы. Само название проекта — это название статьи. Вы можете перевести название на английский или использовать транскрипцию (написание русских слов английскими буквами).

К примеру, название статьи: «Как написать статью для Special». Значит название проекта может быть одним из двух:

  • «how-to-write-an-article-for-special»;
  • «kak-napisat-statyu-dlya-special».

Данное название проекта будет являться URL путем. Ваша статья будет находиться по следующему адресу: https://news.dybka.ru/<проект>/

Публикация статьи

Для публикации статьи вы можете использовать один из двух вариантов.

Первый вариант: отправка пулл-реквеста на репозиторий GitHub.

Второй вариант: напишите автору напрямую. Пришлите проект в качестве zip-архива. Ссылки на соцсети:

  • телеграм
  • вконтакте
  • daniil@dybka.ru

После отправки проекта, ваша статья будет проверена. После проверки будет выдвинуто решение, опубликовать статью или нет.
